A big update focused on working alongside your editor and AI tools.

**New**

- **Seamless live reload** — the page no longer flashes when your file changes, and you never lose your place while reading
- **Follow Edits mode** (View menu) — the preview automatically scrolls to whatever just changed. Perfect for watching an AI assistant write your docs in real time
- **Copy button on code blocks** — hover any code block and copy its source with one click
- **Recent files on the welcome screen**
- **Built-in Help and a sample document** — see everything the app can do, right from the Help menu
- **One-click integration setup** — connect your terminal (`mbv`, now also installable via Homebrew) and AI tools like Claude Code, Cursor or Windsurf from Help ▸ Set Up Integration…
- Opening a file from the terminal brings the window to the front; AI tools keep updating it quietly in the background
- Redesigned, cleaner **Settings** and **Integration** windows

**Fixed**

- Images with accented or non-English filenames (like `obrázek.png`) now display correctly — in the app and in Quick Look
- Clicking an image opens it full size again
- Ordinary dollar amounts ($5 and $10) are no longer mistaken for math formulas
- Table-of-contents links now always jump to the right heading, even for duplicate or formatted headings
- Opening documents via `mdbv://` links from other apps now works reliably
- Many smaller stability and security improvements
