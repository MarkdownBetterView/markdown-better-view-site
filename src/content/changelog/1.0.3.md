Quality-of-life update on top of 1.0.2.

**New**

- **Recent files on the welcome screen** — jump back into your last documents with one click
- **Copy button on code blocks** — hover any code block and copy its source instantly
- **Sample document** (Help ▸ Open Sample Document) — see everything the app can do, including a bundled local image
- **Redesigned Settings and Integration windows** — cleaner, grouped, easier to read
- The `mbv` terminal tool can now also be installed via **Homebrew** (`brew install xxBedy/tap/mbv`)
- Opening a file from the terminal now brings the window **to the front**; AI tools keep updating it quietly in the background

**Improved**

- `mbv` 1.1.0: opens in the foreground by default; use `-b` for silent background updates
