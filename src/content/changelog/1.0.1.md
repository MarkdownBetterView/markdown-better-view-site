First release on the Mac App Store — a fast, native Markdown viewer for macOS.

- Beautiful rendering of GitHub-flavored Markdown: tables, task lists, callouts, Mermaid diagrams and math
- Live reload — edit in any editor, the preview updates on every save
- Themes, dark mode, reading mode, table of contents, in-page search
- Export to PDF and print
- Quick Look previews and Finder thumbnails for your Markdown files
- A welcome window gets you started: drop a file or open one with a click
