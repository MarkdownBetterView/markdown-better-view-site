Markdown Better View 1.0.5 — your default Markdown viewer, plus reliability fixes

Version 1.0.5 · July 13, 2026

Everything since 1.0.3, in one release.

New

  • Set it as your default Markdown viewer — one click at first launch, or any time from Settings
  • Quick Look now works for .mdown and .mkd files — previews and Finder thumbnails, not just .md/.markdown
  • Quick links in Help and About — jump straight to Integrations, the website, or rate the app on the App Store

Improved

  • Follow Edits keeps tracking your changes even right after a theme or appearance switch — no more stopping mid-stream until you restart
  • Faster page loads, plus smoother live updates for large images and Mermaid diagrams
  • Quicker app launch

Fixed

  • A zoom glitch where a reload could briefly double-scale the page
  • Recovering from a rare WebKit hiccup now re-renders your document instead of leaving a blank window

Also new: a direct download. Markdown Better View is now available outside the Mac App Store too — a notarized download with a 14-day trial and automatic updates. See the download page. The Mac App Store remains the simplest way to buy.

Markdown Better View 1.0.3 — recents, code copy & a fresh coat of paint

Version 1.0.3 · June 12, 2026

Quality-of-life update on top of 1.0.2.

New

  • Recent files on the welcome screen — jump back into your last documents with one click
  • Copy button on code blocks — hover any code block and copy its source instantly
  • Sample document (Help ▸ Open Sample Document) — see everything the app can do, including a bundled local image
  • Redesigned Settings and Integration windows — cleaner, grouped, easier to read
  • The mbv terminal tool can now also be installed via Homebrew (brew install xxBedy/tap/mbv)
  • Opening a file from the terminal now brings the window to the front; AI tools keep updating it quietly in the background

Improved

  • mbv 1.1.0: opens in the foreground by default; use -b for silent background updates

Markdown Better View 1.0.2 — smoother live reload, built-in Help

Version 1.0.2 · June 11, 2026

A big quality update focused on working alongside your editor and AI tools.

New

  • Seamless live reload — the page no longer flashes when your file changes, and you never lose your place while reading
  • Follow Edits mode (View menu) — the preview automatically scrolls to whatever just changed. Perfect for watching an AI assistant write your docs in real time
  • Built-in Help — a quick guide to everything the app can do, right in the Help menu
  • One-click integration setup — connect your terminal (mbv command) and AI tools like Claude Code, Cursor or Windsurf from Help ▸ Set Up Integration…

Fixed

  • Images with accented or non-English filenames (like obrázek.png) now display correctly — in the app and in Quick Look
  • Clicking an image opens it full size again
  • Ordinary dollar amounts ($5 and $10) are no longer mistaken for math formulas
  • Table-of-contents links now always jump to the right heading, even for duplicate or formatted headings
  • Opening documents via mdbv:// links from other apps now works reliably
  • Many smaller stability and security improvements
